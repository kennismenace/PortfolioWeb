PRINTING THE WEB


This is a visual research project to investigate the web. What does the web look like translated into printed form? What is the underlying architecture of an interface? How to translate intangible interfaces experiences, such as clicks, scrolls, rollovers, and responsiveness to ink and paper?

"For us, makers, the creative process has transformed into a series of digital clicks and actions. These actions are temporary. Once executed, they disappear from the screen.
We can hardly trace back how things looked when we were creating them. Is there a way we can archive, publish and reproduce these actions?"
- Inez Cox



PROCESS:

1. Analyze the interface architecture.
2. Map out the digital actions and frameworks.
3. Reveal the underlying architecture.
4. Deconstruct and reveal.
5. Exhaustive visual studies.
6. Designed one poster composition based on the visual research and studies.

Purpose - Web Anatomy, Interface Aesthetics, Translation, Exploration, Visual Research
What is it - Printing the Web
Instructor - Jon Sueda, Catherine Schmidt
Exhibition - CCA Hubbell Street Galleries - 2018
