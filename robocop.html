ROBOCOP


What is the smallest building block of typography? Is it the word? Or the letter? The shape of a counter? The space between letterforms? This project was to break apart language into its smallest possible forms and build it back up again. This type specimen comes in full alphabet, numerals, and basic punctuations, in 3 type styles.



PROCESS:

The designs were based on the component pieces from the found objects (stencils). First the designs were hand drawn and then converted into vectors. Lastly, using the completed type specimen to design a typographic composition.

Purpose - Alphabetical System, Type Study, Experiment
What is it - Type Specimen
