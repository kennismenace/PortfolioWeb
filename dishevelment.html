DISHEVELMENT


This project was to generate new forms through modifying existing forms from the browser inspector, by altering the HTML and CSS. Inspect Element is a browser tool that lets the user view and edit the HTML and CSS of a web content. Any edits the viewer make will only appear on their personal screen, and last until the viewer exits out. There are no permanent changes being made to a website, it only allows to adjust for the time being. Due to the nature of the Inspect Element, each altered composition had to be screenshotted before formulating into a book.
